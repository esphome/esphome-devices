## Flashing Procedure

Comes pre-flashed with a customized Tasmota optimized for Home Assistant, OpenHab and Domoticz.  You can easily upload an ESPHome binary via OTA from the built-in web interface.

## Switch Specifications

Voltage: 110V
Load Capacity: 1500W
