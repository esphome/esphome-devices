## GPIO Pinout

| Pin    | Function            |
| ------ | ------------------- |
| GPIO36 | Free GPIO           |
| GPIO39 | Free GPIO           |
| GPIO34 | Free GPIO           |
| GPIO35 | Free GPIO           |
| GPIO4  | IIC_SDA             |
| GPIO5  | IIC_SCL             |
| GPIO16 | RS485_RXD           |
| GPIO14 | RS485_TXD           |
| GPIO33 | RS232_RXD           |
| GPIO32 | RS232_TXD           |

[Additional pinout/design details](https://www.kincony.com/esp32-programmable-keyboard.html)
