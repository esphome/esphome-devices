## GPIO Pinout

| Pin    | Function            |
| ------ | ------------------- |
| GPIO36 | ANALOG_A1           |
| GPIO34 | ANALOG_A2           |
| GPIO35 | ANALOG_A3           |
| GPIO39 | ANALOG_A4           |
| GPIO4  | IIC_SDA             |
| GPIO5  | IIC_SCL             |
| GPIO14 | 1-Wire GPIO         |
| GPIO16 | 433MHz Receiver     |
| GPIO32 | RS485_RXD           |
| GPIO33 | RS485_TXD           |
| GPIO15 | GSM_RXD             |
| GPIO13 | GSM_TXD             |
