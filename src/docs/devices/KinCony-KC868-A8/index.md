## GPIO Pinout

| Pin    | Function            |
| ------ | ------------------- |
| GPIO34 | ANALOG_A1           |
| GPIO35 | ANALOG_A2           |
| GPIO4  | IIC_SDA             |
| GPIO5  | IIC_SCL             |
| GPIO14 | 1-Wire GPIO         |
| GPIO13 | 1-Wire GPIO         |
| GPIO32 | 1-Wire GPIO         |
| GPIO33 | 1-Wire GPIO         |
| GPIO15 | 433MHz Transmitter  |
| GPIO2  | 433MHz Receiver     |

[Additional pinout/design details](https://www.kincony.com/arduino-esp32-8-channel-relay-module-kc868-a8.html)
