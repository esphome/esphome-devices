## GPIO Pinout

| Pin    | Function            |
| ------ | ------------------- |
| GPIO34 | ANALOG_A1           |
| GPIO35 | ANALOG_A2           |
| GPIO39 | ANALOG_A3           |
| GPIO36 | ANALOG_A4           |
| GPIO4  | IIC_SDA             |
| GPIO5  | IIC_SCL             |
| GPIO14 | 433MHz Receiver     |
| GPIO16 | IR Receiver         |
| GPIO33 | RS485_RXD           |
| GPIO32 | RS485_TXD           |

[Additional pinout/design details](https://www.kincony.com/32-channel-esp32-mosfet-board.html)
