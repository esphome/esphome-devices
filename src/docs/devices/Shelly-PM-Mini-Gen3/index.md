Generation 3 of Shelly Mini. With Powermeter.

## GPIO Pinout

| Pin   | Function     |
| ----- | ------------ |
| GPI03 | NTC          |
| GPI06 | BL0942 TX    |
| GPI07 | BL0942 RX    |
| GPI07 | LED          |
| GPI07 | Button       |


## Serial Pinout

The UART Pinout is the same as other Shelly Plus Mini.
