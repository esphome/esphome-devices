## GPIO Pinout

| Pin    | Function                           |
| ------ | ---------------------------------- |
| GPIO16 | TX RS485                           |
| GPIO04 | RX RS485                           |
| GPIO02 | flow_controll RS485                |
| GPIO0  | WS2812B RBG Status LED             |
| GPIO32 | mcp23s08_hub                       |
