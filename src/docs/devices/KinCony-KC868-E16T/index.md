## GPIO Pinout

| Pin    | Function            |
| ------ | ------------------- |
| GPIO36 | ANALOG_A1           |
| GPIO35 | ANALOG_A2           |
| GPIO34 | ANALOG_A3           |
| GPIO39 | ANALOG_A4           |
| GPIO16 | IIC_SDA             |
| GPIO15 | IIC_SCL             |
| GPIO14 | 1-Wire GPIO         |
| GPIO4  | 433MHz Receiver     |
| GPIO2  | Buzzer              |
| GPIO12 | LED                 |
