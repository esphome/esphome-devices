## GPIO Pinout

| Pin    | Function            |
| ------ | ------------------- |
| GPIO36 | ANALOG_A1           |
| GPIO34 | ANALOG_A2           |
| GPIO39 | ANALOG_A3           |
| GPIO35 | ANALOG_For_74HC4067 |
| GPIO4  | IIC_SDA             |
| GPIO5  | IIC_SCL             |
| GPIO14 | 1-Wire GPIO         |
| GPIO16 | 433MHz Receiver     |
| GPIO32 | 74HC4067_S0         |
| GPIO33 | 74HC4067_S1         |
| GPIO13 | 74HC4067_S2         |
| GPIO16 | 74HC4067_S3         |
