# M5Stack NanoC6

M5Stack NanoC6 is a tiny USB-powered devkit based on the ESP32-C6.


## GPIO Pinout

| Pin    | Function        |
| ------ | --------------- |
| GPIO1  | Grove I2C SCL   |
| GPIO2  | Grove I2C SDA   |
| GPIO3  | Infrared LED    |
| GPIO7  | Blue LED        |
| GPIO9  | Button          |
| GPIO19 | RGB Power       |
| GPIO20 | RGB LED         |
