## GPIO Pinout

| Pin    | Function       |
| ------ | -------------- |
| GPIO01 | CSE7766 Tx     |
| GPIO03 | CSE7766 Rx     |
| GPIO12 | Button 1       |
| GPIO13 | LED 1 inverted |
| GPIO14 | Relay 1        |
