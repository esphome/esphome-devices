## GPIO Pinout

| Pin    | Function                           |
| ------ | ---------------------------------- |
| GPIO36 | ANALOG_A1                          |
| GPIO39 | ANALOG_A2                          |
| GPIO34 | ANALOG_A3                          |
| GPIO35 | ANALOG_A4                          |
| GPIO4  | IIC_SDA                            |
| GPIO16 | IIC_SCL                            |
| GPIO5  | DS18B20/DHT11/DHT21/LED strip-1    |
| GPIO14 | DS18B20/DHT11/DHT21/LED strip-2    |
| GPIO23 | ETH_MDC                            |
| GPIO18 | ETH_MDIO                           |
| GPIO33 | RS485_RXD                          |
| GPIO32 | RS485_TXD                          |
| GPIO13 | GSM_RXD                            |
| GPIO15 | GSM_TXD                            |

[Additional pinout/design details](https://www.kincony.com/forum/forumdisplay.php?fid=41)
