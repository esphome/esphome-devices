## GPIO Pinout

| Pin    | Function   |
| ------ | ---------- |
| GPIO1  | Status LED |
| GPIO20 | Button     |
| GPIO4  | Relay      |
| GPIO7  | BL0937 CF  |
| GPIO3  | BL0937 CF1 |
| GPIO5  | BL0937 SEL |
| GPIO6  | Relay Led  |
