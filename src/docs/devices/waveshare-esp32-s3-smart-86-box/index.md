# Waveshare ESP32-S3 Smart 86 Box

The Waveshare ESP32-S3 Smart 86 Box is a wall-panel style ESP32-S3 device with a 480 x 480 touch display, microphone, speaker, Wi-Fi, Bluetooth, and Home Assistant voice assistant support.

This configuration provides display output, touch input, audio input and output, wake word support, Bluetooth proxy support, OTA updates, diagnostics, and Home Assistant voice assistant integration.
