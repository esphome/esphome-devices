## GPIO Pinout

| Pin    | Function            |
| ------ | ------------------- |
| GPIO36 | ANALOG_A1           |
| GPIO39 | ANALOG_A2           |
| GPIO34 | ANALOG_A3           |
| GPIO35 | ANALOG_A4           |
| GPIO4  | IIC_SDA             |
| GPIO5  | IIC_SCL             |
| GPIO14 | 1-Wire GPIO         |
| GPIO16 | 433MHz Receiver     |
| GPIO32 | RS485_RXD           |
| GPIO33 | RS485_TXD           |
| GPIO13 | GSM_RXD             |
| GPIO15 | GSM_TXD             |
| GPIO2  | Buzzer              |
| GPIO12 | WS2812B LED         |
