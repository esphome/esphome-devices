## GPIO Pinout

| Pin    | Function                   |
|--------|----------------------------|
| GPIO02 | Blue LED (Inverted: true)  |
| GPIO05 | HLW8012 CF Pin             |
| GPIO12 | HLWBL SELi Pin             |
| GPIO13 | Push Button                |
| GPIO14 | HLWBL CF1 Pin              |
| GPIO15 | Relay                      |
